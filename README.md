# Multiple Linear Regression – Startup Profit Prediction

## 📌 Overview

This project demonstrates how to build a **Multiple Linear Regression** model to predict startup profits based on business expenditure data.

The model uses multiple independent variables including **R&D Spend**, **Administration**, **Marketing Spend**, and **State** to estimate a startup's profit.

---

## 🎯 Objective

Develop a machine learning model capable of predicting startup profit using multiple features while demonstrating the complete regression workflow using Scikit-learn.

---

## 📂 Dataset

Dataset: **50_Startups.csv**

### Features

- R&D Spend
- Administration
- Marketing Spend
- State

### Target

- Profit

---

## 🛠 Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## 📈 Machine Learning Workflow

- Import libraries
- Import dataset
- Encode categorical variables
- Split dataset into training and testing sets
- Train Multiple Linear Regression model
- Predict test results
- Evaluate model performance using R² Score
- Display regression coefficients
- Predict profit for a new startup

---

## 📊 Model Performance

| Metric | Value |
|---------|-------|
| Algorithm | Multiple Linear Regression |
| R² Score | **0.935** |
| Dataset | 50 Startups |

The trained model explains approximately **93.5%** of the variance in startup profit.
